### 交换一个整数的Bit位的奇数位和偶数位

```c
// 将奇数位的数拿出来左移一位，偶数位拿出来右移一位，然后加一起就是转换后的数
// 拿出偶数位的方法：跟10101010 10101010 10101010 10101010按位与一下
// 拿出奇数位的方法：跟01010101 01010101 01010101 01010101按位与一下
#define SWAP_BIT(a) a = (((a&0x55555555)<<1)+((a&0xaaaaaaaa)>>1))

int main()
{
	int a = 0;
	scanf("%d", &a);

	SWAP_BIT(a);
	printf("%d", a);

	return 0;
}
```
